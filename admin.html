Guía de Configuración para n8n

Conectar Verocenter con n8n te permite automatizar tareas complejas, tales como agregar clientes a una base de datos (Google Sheets/Notion) o enviar un recordatorio automático por WhatsApp.

1. Crear el Workflow en n8n

  • Agrega un nodo Webhook.
  • Establece el método HTTP como POST.
  • Copia la Production URL del nodo Webhook de n8n y pégala arriba en el campo "URL del Webhook de n8n".
  • (Opcional) En el nodo Webhook de n8n, configura autenticación (Header Auth) e introduce las credenciales arriba en "Nombre Cabecera Auth" y "Token".

2. Estructura del Payload Enviado (JSON)

Cuando un cliente solicita una reserva, la web envía automáticamente un payload POST con la siguiente estructura:

{
  "event": "booking_requested",
  "timestamp": "2026-07-01T00:30:00.000Z",
  "client": {
    "name": "Valentina Gómez",
    "phone": "+56912345678"
  },
  "booking": {
    "service_id": "acrilico-m",
    "service_name": "Acrílico - Largo M",
    "price": "27.000",
    "duration_minutes": "130",
    "date": "2026-07-05",
    "time": "15:00",
    "notes": "Prefiero acabado matte."
  },
  "automation": {
    "recipient_phone": "56975116975",
    "channel": "whatsapp_api_direct",
    "message_payload": "..."
  }
}
